How to use Package Helper API

1. Install PackageHelper.apk :
> adb install PackageHelper.apk
Source Code: https://github.com/zzhouj/PackageHelper

2. Start web service of PackageHelper:
> adb shell "am startservice -n com.android.packagehelper/.WebService"

3. Forword web service tcp port to local:
> adb forward tcp:8787 tcp:8787

4. Access web service of PackageHelper:

  • Home (this page): http://localhost:8787/
  • Package infos (JSON array): http://localhost:8787/list
  • Package icon (PNG): http://localhost:8787/icon?p=com.android.mms

A. Uninstall PackageHelper.apk:
> adb uninstall com.android.packagehelper
